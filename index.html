Model equations & assumptions

ISA Atmosphere

T(h) = 288.15 − 0.0065·hρ = P₀·(T/T₀)^(g/RL) / (R·T)

Override bypasses ISA.

Drag Polar

CL = 2W / (ρV²S)CD = CD₀ + CL²/(π·e·AR)

Power Chain

P_air = D·VP_total = P_air/η + P_hotel + P_payload

Energy Budget

E_nom = m_bat × SEE_usable = E_nom · f_usable · f_Pk

Peukert

f_Pk = (E_nom / P_total)^(n−1)

n = 1 → ideal. Ref = 1-hour discharge.

Climb / Descent

P_climb = P_level + m·g·Vc/ηP_desc = f_d·(P_air/η) + P_np

Wind Model

V_gs = max(0, V_air − V_wind)

Per-leg winds used in profile; single headwind for top-level KPIs.

Payload Drop

Post-loiter segments use return_payload_mass_kg — all derived quantities update.

SM = V / V_stall ≥ 1.3
