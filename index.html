UAV Mission Performance Estimator

Fixed-wing preliminary performance and segmented mission analysis — browser-side calculations, no server required.

Methodology & model assumptions

Atmosphere — ISA troposphere (h ≤ 11 000 m)

T(h) = 288.15 − 0.0065·h

ρ(h) = p₀·(T/T₀)^(g₀/RL) / (R·T)

R = 287.05 J/kg/K, g₀ = 9.80665 m/s²

An air-density override bypasses ISA and uses the entered ρ directly.

Aerodynamics — parabolic drag polar

CL = 2W / (ρV²S) — level flight (L = W)

CD = CD₀ + CL² / (π·e·AR)

D = ½ρV²S·CD

Compressibility not modelled. Lift constraint applies per-segment at each altitude.

Power chain

P_air = D·V

P_prop = P_air / η_total

P_total = P_prop + P_hotel + P_payload

Hotel and payload loads are constant; loiter payload load replaces cruise payload load during loiter segments.

Energy budget

E_nom = m_bat × SE

E_usable = E_nom × f_usable × f_Peukert

E_mission = E_usable − E_reserve

Endurance = E_mission / P_total

Peukert correction

f_Peukert = (E_nom / P_total)^(n − 1)

n = 1 → ideal (default). Typical Li-Po: n ≈ 1.02–1.08. Applied per speed-sweep point, so operating points shift slightly when n > 1. Reference discharge is the 1-hour rate.

Climb segment

P_climb = P_level + m·g·Vc / η_total

Drag evaluated at cruise speed (small-angle approx, valid for γ < ~10°). Climb speed is not independently optimised — set cruise speed to your intended climb speed for fidelity.

Descent segment

P_descent = f_descent × (P_air / η_total) + P_np

f_descent scales propulsive power relative to cruise. P_air anchored to cruise speed; descent rate only affects duration. Adjust f_descent to represent different throttle strategies.

Wind model

V_ground = max(0, V_air − V_wind)

Top-level KPIs use a single general wind speed (conservative one-leg headwind estimate). Segmented Mission profile uses separate per-leg wind speeds for accurate round-trip analysis. The asymmetric time penalty (slow outbound, fast return) is visible in the segment table.

Operating point search

Optimal speeds found by numerical search over 120-point V_stall → 40 m/s sweep.

Constrained to V ≥ 1.3·V_stall for operating-point selection. Sweep results are memoised per unique parameter set within each render call. Best wind-adjusted range maximises V_ground × endurance at the given leg wind.

Payload drop

If Return payload mass is set in the profile, aircraft mass is reduced from that value for all post-loiter segments (return & descent). Weight, stall speed, drag and all derived quantities update accordingly.

Leave blank for a standard fixed-mass mission.
